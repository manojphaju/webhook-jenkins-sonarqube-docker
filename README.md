# Jenkins CI/CD Pipeline - SonarQube, Docker, Github Webhooks on AWS

# Project Description
The pipeline is deployed to the docker after scanning the code through SonarQube and is hosted on AWS. The pull and push are triggered using the webhoooks to automate the process.
